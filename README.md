## About This Project

This is a professional, responsive website built for a South African law firm. It showcases their services, team, and contact information. The site is live and serves as a public-facing platform for the firm.

## Key Features

- Clean, professional design for a legal practice
- Fully responsive for mobile and desktop
- 5+ pages: Home, About, Services, Team, Contact
- Integrated location map and contact details

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Git & GitHub
- FTP and DNS management (deployed to a Linux server)

## Live Site

Visit the live website here: [http://mangalaw.co.za](http://mangalaw.co.za)

## What I Learned

This project taught me:

- How to coordinate with a client and manage requirements
- How to deploy a website via FTP and configure DNS
- How to handle third-party IT providers
- The importance of adapting to server constraints (Linux vs Windows)
- How to debug 403, 404, and raw code display issues

## How to View the Code

1. Clone this repository
2. Open `index.html` in any browser
