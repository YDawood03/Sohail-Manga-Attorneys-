 About This Project

This is a professional, responsive website built for a South African law firm. It showcases their services, team, and contact information. The site is live and serves as a public-facing platform for the firm.

The project involved end-to-end delivery: from client consultation and design to deployment on a Linux server via FTP, including DNS configuration and coordination with a third-party IT provider.

 Key Features

- Clean, professional design tailored for a legal practice
- Fully responsive for mobile, tablet, and desktop
- 5+ pages: Home, About, Services, Team, Contact
- Integrated location map and contact details
- Optimised for fast loading and accessibility

 Technologies Used

- HTML5 – Structure and content
- CSS3 – Styling, layout, and responsiveness (Flexbox, Grid, Media Queries)
- Git & GitHub – Version control and portfolio hosting
- FTP & DNS – Deployment to a live Linux server

 What I Learned

This project was a valuable real-world experience that taught me:

- Client Communication – How to gather requirements and manage expectations.
- Deployment & Hosting – How to upload a site via FTP, configure DNS, and work with a Linux server.
- Third-Party Coordination – How to liaise with external IT providers and navigate stakeholder relationships.
- Adaptability – How to pivot from ASP.NET to a simpler HTML/CSS solution when server constraints arose.
- Debugging – How to troubleshoot and resolve 403, 404, and raw code display errors in a live environment.
